<h1 align="right">Rehab360 – פרויקט תכנות ווב (Desktop-first + Responsive)</h1>

<p>
<b>Rehab360</b> היא מערכת ווב לניהול תהליך שיקום דיגיטלי עם שני תפקידי משתמש:
<b>Patient</b> ו-<b>Physiotherapist</b>.
</p>

</div>

---

##  מה יש בפרויקט
- **Welcome**: בחירת תפקיד
- **Patient**: דשבורד + תוכנית טיפול + טופס דיווח תרגיל
- **Physiotherapist**: דשבורד + מסך פרטי מטופל

---

##  טכנולוגיות
- HTML / CSS (עיצוב אחיד ורספונסיבי)
- JavaScript + jQuery (אינטראקציות, ולידציות, שליחה ב־AJAX)
- PHP + MySQL (שמירת דיווחים)

---

##  איך מריצים
**כתובת השרת :** `shanimi2.mtacloud.co.il`

**מקומית (Front בלבד):**
- לפתוח `index.html` בדפדפן.

**מקומית (כולל שמירה ל־DB):**
- להריץ דרך שרת שתומך PHP + MySQL
- לעדכן פרטי התחברות ב־`includes/exercise_report.php`

---

##  מבנה תיקיות (בקיצור)
- `index.html`
- `includes/` – עמודי HTML + `exercise_report.php`
- `css/` – קבצי עיצוב
- `js/` – קבצי JavaScript/jQuery
- `images/` – תמונות/אייקונים
